### Difference services(server) running in Data center
**DNS(Domain Name System) server**
- It convert Domain name into IP address and also reverse.
- It is manage by the serives provider or maybe the local organiazation.

**DHCP(Dynamic Host Configuration Protocol)**
- It is used when we connect any device to the network.
- In the large company they has multiple DHCP servers if one server is unavailble it will connect to the other server.

**File sharing**
- It will provide a way to centrally store information to access the file by everyone.
- To share the file to the peoples wthin the organication or outside of the organization.
  - we can use the standard type of file service.
    - [ ] SMB (Server Message Block) used in Windows.
    - [ ] AFP (Apple Filing Protocol) used in Mac OS.
    - There are many other different standards for sharing the file it depands on the app and the OS which is used.
    - Usaly the operating system tends to hide those protocols.
    - So use will see the File management front end options like drag, drop, rename, modify, etc.. in the folder.

**Printer Service**
- It is used to connect the printer to the system the print the information on the paper.
- Sometimes it will use the separate computer which will also run the print service application or it may be build into the printer itself, It may be a network card it will provide interface to the network and the print serivcwe software.
- It will provide the different protocols in the network like
  - [ ] SMB (Server Message Block)
  - [ ] IPP (Internet Printing Portocol)
  - [ ] LPD (Line Printer Daemon)
