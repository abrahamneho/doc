# Laptop Hardware Basics (Core 1 – Section 1.1)

## 1. Laptop Batteries

There are two types of batteries used in laptops:

### Removable Batteries

* These batteries can be easily removed by users.
* There is usually a latch or button to release the battery.
* Users can replace the battery without technical help.

### Built-in Batteries

* These batteries cannot be removed easily by users.
* Replacing them usually requires opening the laptop.
* A technician is usually needed for replacement.

### Types of Batteries Used in Laptops

* **Lithium-ion (Li-ion) batteries**
* **Lithium-ion Polymer (Li-Po) batteries**

Both battery types do not have the traditional *memory effect* found in older battery technologies.
This means they can be recharged at any time without significantly affecting their overall capacity.

---

## 2. Laptop Keyboard

### Common Keyboard Problems

* Some keys may stop working.
* Keys may become loose or damaged.

### How to Identify the Problem

* To test if a key is faulty, you can try connecting an external keyboard.
* If the external keyboard works fine, the issue is with the laptop keyboard.

### Fixing a Key Problem

* Sometimes only a single key needs replacement.
* Carefully remove the keycap.
* Replace it with a new one.
* Handle the keyboard carefully to avoid damaging other keys.
* Always follow the manufacturer’s instructions when replacing keys.

---

## 3. Laptop Memory Modules and Storage

### Memory Modules

* Laptops commonly use **SO-DIMM** memory modules.
* These modules are usually easy to remove and replace.
* Some laptops have soldered RAM, which cannot be upgraded.
* In such cases, the entire motherboard must be replaced.

### Memory Upgrade Process

* Open the back panel of the laptop.
* Insert the new RAM into the available slot.
* Ensure it is properly seated before closing the laptop.

### Storage Devices

There are two main types of storage hardware:

#### HDD (Hard Disk Drive)

* Used in older laptops and desktops.
* Uses spinning disks to store data.
* Slower compared to SSDs.

#### SSD (Solid State Drive)

* Used in most modern laptops.
* Much faster than HDDs.
* Common interfaces include **SATA** and **M.2 NVMe**.

### Data Transfer Between Drives

* Install both drives (old and new) in the system.
* Use cloning software or data transfer tools.
* Transfer data from the old drive to the new drive.

---

## 4. Wireless Connections and Bluetooth

Modern laptops usually include built-in wireless connectivity.

### Wi-Fi (802.11)

* A wireless local area network (WLAN) technology.
* Used for internet connectivity without cables.

### Bluetooth

* A short-range wireless communication technology.

* Used to connect devices like:

  * Mouse
  * Keyboard
  * Headphones
  * Smartphones

* Usually built into the motherboard.

* Older laptops and desktops may require external adapters.

---

## 5. Biometrics in Laptops

Biometric hardware is used for secure authentication.

### Common Biometric Devices

#### Webcam (Camera)

* Used for facial recognition.
* Enables Face ID login.

#### Fingerprint Reader

* Used for fingerprint authentication.
* Provides quick and secure login access.

## 6. NFC in Phones and Smartwatches

**NFC (Near Field Communication)** is a short-range wireless technology.

### Uses of NFC

* Contactless payments
* Access cards
